# Number Guessing Game

A simple console-based Number Guessing Game made in C.

## Current Version V1.0

### Features
- Takes player's name as input
- Greets the player
- Displays game rules
- Number guessing system
- "Very Close +" and "Very Close -" hints
- "Close +" and "Close -" hints
- Win detection
- Play Again (Y/N) option


## Future Updates
- Attempt Counter
- Score System
- Difficulty Levels (Easy / Medium / Hard)
- Random Number Generation
